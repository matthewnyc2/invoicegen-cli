# InvoiceGen: Invoice Management Dashboard

## Vibe
Professional business dashboard with Brutalist Command aesthetic. Dark background, cyan header accents, green status indicators. High-density information display optimized for revenue management.

## Design System
- **Background**: `#0e0e0e` (dark black)
- **Header Accent**: `#00fbfb` (cyan blue)
- **Primary Accent (Positive)**: `#9fff88` (neon green)
- **Secondary Accent (Alerts)**: `#44a5ff` (sky blue)
- **Text**: High contrast white with subtle gray hierarchy
- **Status Badges**: Color-coded (green=Paid, yellow=Sent, red=Overdue)

## Page Structure
Display "Invoices" table with columns:
- **ID**: Numeric invoice identifier
- **Invoice #**: Formatted as INV-YYYY-NNNN (e.g., INV-2026-0001)
- **Client**: Client company name
- **Total**: Dollar amount (right-aligned)
- **Status**: Color-coded badge (Paid, Sent, Overdue)
- **Created**: Invoice creation date
- **Due Date**: Payment due date

## Layout Details
- Clean table layout with ghost borders (low opacity)
- Header with cyan accents
- Row alternation for readability
- Compact spacing for high information density
- Professional business appearance
- Status badges with clear visual hierarchy
- All amounts displayed in currency format

# InvoiceGen: Revenue Report Dashboard

## Vibe
Financial analytics dashboard with revenue visualization. Terminal-inspired bar charts using Unicode block characters. Professional business analytics interface.

## Design System
- **Background**: `#0e0e0e` (dark black)
- **Header**: `#00fbfb` (cyan accent)
- **Chart Bars**: Gradient from `#9fff88` (green) to `#44a5ff` (blue)
- **Text**: White with gray secondary text
- **Highlights**: `#00fbfb` (cyan for important numbers)

## Page Structure
**Report Title**: "Revenue Report" or "Monthly Revenue" (centered, large)

**Monthly Summary Section**:
- Month labels on left (January through December)
- Horizontal bar charts with Unicode block characters
- Dollar amounts at end of each bar
- Alternating row backgrounds for readability

**Statistics Section**:
- Total revenue (highlighted in green)
- Average monthly (gray text)
- Highest month (cyan accent)
- Lowest month (gray text)

**Quarterly Summary** (optional lower section):
- Q1, Q2, Q3, Q4 bars
- Larger scale than monthly view
- Same color scheme

## Layout Details
- Clean two-column layout (labels + charts)
- Unicode blocks for bar visualization
- Right-aligned amounts for easy scanning
- Professional financial dashboard appearance
- High information density
- Clear hierarchy using color and size
